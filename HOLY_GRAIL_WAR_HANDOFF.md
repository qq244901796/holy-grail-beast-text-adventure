# 圣杯战争文字冒险交接文档

## 1. 目标与定位

当前这套玩法是基于 ai-text-adventure 原始引擎，改造成的 Fate / FGO 风格圣杯战争文字冒险。

设计目标：

- 通过 bot 对话游玩，而不是网页单机为主
- 玩家扮演御主，随机召唤一名从者
- 世界观尽量接近正统圣杯战争，而不是校园怪谈或泛奇幻冒险
- 玩法以“回合推进 + 状态管理 + 自动战斗 + 分支结局”为核心
- 强调“随机召唤不同从者会带来不同体验”

当前版本已经不再是简单换皮，而是：

- 世界观数据已换成圣杯战争语境
- 英灵池已接入
- 自动战斗已接入
- 选项重复问题已做一轮优化
- 结局已按职阶/契约倾向做初步分化

## 3. 当前已完成内容

### 3.1 世界观与行动池

engine-data.js 已改成圣杯战争专用数据，包含：

- 冬木风格地点
- 白天/夜晚气氛
- 圣杯战争行动文案
- 圣杯战争事件池

当前行动核心是：

- 搜索灵脉
- 试探御主
- 术式布防
- 夜间追猎
- 补充魔力
- 与从者对话
- 主动强袭
- 搜集情报
- 强化据点

### 3.2 英灵池

heroic-spirits.js 已接入大池英灵，按七职阶组织。

当前英灵来源是直接参考 Fate / FGO 常见从者，不再做“影射型原创英灵”。

已经实现：

- 随机召唤
- 每名从者的基础标签
- 职阶基础 profile
- 开局召唤台词
- traits 对战斗、行动、忠诚等的影响

### 3.3 开局节奏分化

当前有开局契约 profile：

- unstable：危险契约
- predator：速攻猎杀
- 	actician：谋略展开
- guardian：稳守护航
- sovereign：王者压场
- standard：常规推进

这些会影响：

- 初始资源
- 前两回合行动倾向
- 早期推进速度
- 暴露度
- 某些坏结局触发概率

### 3.4 自动战斗

当前战斗设计是：

- 玩家不手搓详细技能
- 玩家通过行动与前置选择影响战局
- 系统自动结算遭遇战
- 返回战报

当前已接入：

- 职阶克制
- trait 加成
- 宝具充能
- 令咒兜底
- 敌方随机英灵
- 早期暴死
- 自动战报

### 3.5 重复优化

已经做了一轮重复控制：

- 最近回合动作降权
- 选项顺序打散
- preview 不重复
- success/fail flavor 不重复
- 事件池最近事件不重复

所以现在不会像最初一样连续几回合总是同一选项排第一、同一句文案反复刷。

### 3.6 逻辑差异化

已经不是纯文案差异，而是“同一动作在不同从者下效果不同”。

例如：

- Saber 更吃对话/观察
- Archer 更吃观察/潜行
- Lancer 更容易用推进换风险
- Rider 更擅长探索/交涉
- Caster 更擅长术式/据点
- Assassin 更擅长潜行/套情报
- Berserker 推进最猛，但也更容易额外消耗

同时 profile 也会额外叠加收益或惩罚。

### 3.7 结局分化

当前已经把结局从“统一模板”改成“按职阶/契约倾向分化”。

已覆盖：

- 御主死亡
- 精神崩溃
- 从者退场
- 开局暴死
- 真相结局
- 胜利结局
- 契约破裂结局
- 败退结局
- 污染失控结局

但目前还是“半模板化分化”，不是大量从者专属独立结局。

## 6. 当前问题与不足

### 6.1 仍然偏模板化

虽然已经比最初强很多，但当前仍然是：

- 大英灵池 + 标签驱动
- 少量专属事件
- 少量专属战报

不是“每个高人气从者都有完整独立剧情线”。

### 6.2 结局虽然分化了，但还不够深

目前是：

- 职阶级别分化
- profile 级别分化

还没有做成：

- 阿尔托莉雅专属结局
- 金闪闪专属结局
- 梅林专属结局
- 摩根专属结局
- 玉藻前专属结局

这种“点名式独立收束”。

### 6.3 事件池仍可继续扩

目前虽然比最初丰富很多，但长期游玩仍会感觉：

- 某些中期事件开始进入模板循环
- observe / talk / craft / sneak / gamble 这些大类仍然需要更多细分事件

### 6.4 宿敌战报还不够深入

现在 SPECIAL_RIVAL_REPORTS 已恢复成可用中文，但整体仍偏保守。

可以继续扩成：

- 更多宿敌对位
- 不同胜负幅度下不同文本
- 特定从者对特定敌人有额外逻辑影响

### 6.5 终局触发还偏快

当前有些局会比较快冲到：

- progress >= 9
- 或短回合内直接进终局

这会导致部分局体验偏短。

如果后续要优化，可以考虑：

- 拉高一部分终局阈值
- 或在中盘增加一个“强制对抗层”，让终局更有过渡

- 工房争夺

- 宝具类型差异

把当前 late-game 的 boss 事件做厚一点：

- 圣杯污染

- 伪圣杯 / 黑化路线
